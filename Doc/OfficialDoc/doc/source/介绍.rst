====
介绍
====
cuber是一个集成学习框架，主要用于各种集成学习算法的开发，具有灵活动态可配置的特点。cuber主要由三大模块组成，控制引擎、计算引擎和调度引擎。控制引擎依托于networkx的有向无环图技术，需要以计算引擎和调度引擎为基础，运行时加载。计算引擎支持Ray,Dask；调度引擎支持Kahn算法。

**主要技术**：

	* __new__技术
	* __init__技术
	* __prepare__技术
	* __call__技术
	* type元类
	* 描述符技术
	* Mixin
	* 装饰器技术
	* namedtuple
	* Networkx
	* FSM-transitions
	* 运算符重载
	* __slots__技术
	* Ray
